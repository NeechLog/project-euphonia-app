# Project Setup Guide

## Prerequisites
- Python 3.x
- UV package manager

## Installation

1. **Navigate to the API directory**:
   ```bash
   cd api
   ```

2. **Synchronize dependencies**:
   ```bash
   uv sync
   ```

## Running the Application

1. **Activate the virtual environment**:
   ```bash
   source api/.venv/bin/activate
   ```

2. **Start the application**:
   ```bash
   python api/app_dia.py
   ```
